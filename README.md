Refinery Management System

A simple and interactive web app made to manage Refinery production and inventory — all in one dashboard.  
Built completely using **HTML, CSS, and JavaScript**, this project shows how a refinery system can monitor production, manage stock levels, and visualize performance through charts.

What This Project Does

This system is designed for three main user roles:
- Operator: Can update daily production and stock levels.
- Manager: Can approve supplier orders and generate reports.
- Admin: Can manage users and check audit logs.

Features

- Login-based Access — Each user gets their own dashboard view.  
- Production & Inventory Charts — Visualized using Chart.js for better insights.  
- Real-time Updates — Instantly updates charts and values when data changes.  
- Role-based Dashboard — Operator, Manager, and Admin have unique features.  
- Modern UI Design — Clean layout with dark header and optional dark mode ready.

 Tech Stack

- Frontend: HTML, CSS, JavaScript (Vanilla JS)
- Charts: Chart.js
- Editor Recommended: Visual Studio Code
